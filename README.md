# DotEnv Switch

## Switch between environmental values in your .env file

### Usage

1. OPTIONAL. Install the package

```bash
npm i -g dotenv-switch
```

```bash
yarn global add dotenv-switch
```

2. Execute the command

```bash
dotenv-switch
```

(If you didn't install the package globally)

```bash
npx dotenv-switch
```
